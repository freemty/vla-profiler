Next Steps

三个方向

A Algorithmic Reduction

观察 DiT attention pattern → 发现 sparsity → 设计 sparse attention / step caching / token pruning

类比: FastVideo VSA (temporal sparse attn) / AutoGaze (4-100x token pruning, CVPR'26) / step distillation

切入: 我们已有 exp01b/05a attention heatmap + exp04a per-step timing → 直接观察哪些 step/token 可以跳

B Pipeline Bubble Mining

在 LingBot-VA / Rhoda 的真实闭环 pipeline 里找"气泡"：E→V→A 串行等待、KV cache 更新开销

类比: DistServe PD disagg / Rhoda Leapfrog Inference (video gen ∥ action exec)

切入: exp04d 闭环数据 (server log timing) → 量化 compute_kv_cache 开销 → overlap 可行性分析

C Computation Ablation

Fast-WAM skip video → 94.5%。哪些计算可以去掉仍然 work？

消融: 去 video denoise / 减 action steps / 去 cross-attn / 减 DiT layers → quality degradation curve

切入: Fast-WAM 5→1 step sweep (exp04c) + Cosmos step sweep (exp09a) 已有 → 扩展到 layer/head ablation

共同前提: profiling 数据告诉我们在哪里砍。10 模型 × E/C/A breakdown = 砍刀的瞄准镜。

Recent Industry Progress

π0.7 PI · 2026-04

从 "model" 变 "pipeline system"。4 组件: High-Level Policy + World Model (subgoal image) + VLA 主干 + Action Expert。Compositional generalization 首现。
信号: model 是 system 的一部分，不是全部。

GEN-1 Generalist AI · 2026-04

99% params from scratch (非 VLM finetune)。500K+ hr 穿戴 pretrain + 1hr task data → 99% 成功率 + 3x faster。
Pete Florence blog: "GEN-1 is a system. Beyond World Models & VLAs."

GENE-26.5 Genesis AI · 2026-05

Flow matching + 自研 20-DoF dexterous hand + $105M seed。3ms control loop。
赌注: 硬件精度 (hand) > 数据规模。

Rhoda DVA Rhoda AI · 2026-03

Direct Video Action: causal video from scratch → IDM (inverse dynamics) 提取 action。Leapfrog Inference 隐藏 video gen 延迟。$450M Series A, $1.7B val。
赌注: 物理理解在 video 里，不需要显式 policy。

共识: 四家都不报 inference Hz。Model ≠ System。Profiling gap 横跨整个行业。
